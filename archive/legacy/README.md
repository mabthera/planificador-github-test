# Versiones historicas

Esta carpeta conserva versiones anteriores del planificador.

La version activa del proyecto es `index.html` en la raiz, basada en `planificador_v2.html`.

Archivos conservados:

- `planificador.html`: version inicial.
- `planificador-copy.html`: copia intermedia.
- `planificador_v22.html`: variante posterior a v2.
- `planificador_v3.html`: variante posterior, no marcada como final.

Estos archivos se mantienen para consulta historica y comparacion. No deberian usarse como entrada principal del proyecto.
